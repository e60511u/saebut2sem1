## SAE Parking – objectif du projet

Cette application web propose une gestion simple d'un parking :
- authentification (inscription, connexion, déconnexion) avec gestion des favoris et des véhicules liés aux comptes utilisateurs ;
- visualisation de la carte du parking et réservation/gestion des places ;
- espace utilisateur pour modifier les préférences et paramètres personnels ;
- interactions avec une base MySQL via un backend PHP organisé en modèles/contrôleurs/vues.

L'objectif est de fournir une démonstration fonctionnelle pour la SAE : montrer l'architecture MVC en PHP, la persistance des données et un parcours utilisateur complet autour du parking.
